# Exercice 2 : Jeu du Nombre à Deviner

## Description
Ce programme est un petit jeu interactif développé en Java. L'ordinateur choisit un nombre aléatoire entre 1 et 100, et l'utilisateur doit le trouver en moins de 10 essais. 

## Objectifs de l'exercice
L'objectif était de mettre en pratique les notions suivantes :
- Génération de nombres aléatoires avec `Math.random()`.
- Utilisation de la classe `Scanner` pour lire les entrées clavier.
- Mise en place d'une boucle `for` pour limiter le nombre de tentatives.
- Utilisation de structures conditionnelles (`if`, `else if`, `else`) pour guider l'utilisateur.

## Fonctionnement du jeu
1. Le programme génère un nombre secret.
2. À chaque tour, l'utilisateur saisit un nombre.
3. Le programme indique si le nombre secret est **plus grand** ou **plus petit**.
4. Si l'utilisateur trouve le nombre, il gagne immédiatement.
5. S'il atteint 10 essais sans succès, il perd la partie.

## Instructions
- Compiler et exécuter le fichier `Main.java`.
- Suivre les instructions affichées dans la console pour jouer.
